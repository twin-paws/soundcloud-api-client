# Security Policy

## Supported Versions

Only the current major version receives security updates.

| Version | Supported |
| ------- | --------- |
| 1.x     | ✅        |
| < 1.0   | ❌        |

## Reporting a Vulnerability

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, email **twinpawsai@icloud.com** with the following details:

- Description of the vulnerability
- Steps to reproduce
- Affected version(s)
- Potential impact
- Any suggested fix (optional)

## Response Timeline

- **Acknowledgment**: within 48 hours
- **Fix**: best effort, depending on severity

Thank you for helping keep soundcloud-api-ts safe.
